# Support

Use [GitHub Issues](https://github.com/alexmorris10x/gpt-language-tutor/issues) for reproducible repository bugs and focused feature requests. Include the Codex version or environment, the relevant lesson or workflow, expected behavior, and a minimal redacted example.

This project is maintained on a best-effort basis with no guaranteed response time. Personal tutoring, curriculum guarantees, and recovery of private study records are outside the support boundary.

Do not post learner records, voice transcripts, account data, or vulnerability details in public issues. Report security problems through [SECURITY.md](SECURITY.md).
