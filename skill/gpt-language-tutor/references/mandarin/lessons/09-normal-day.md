# Lesson 09 — A Normal Day

## Language set

| Chinese | Pinyin | English | Role |
|---|---|---|---|
| 我……点…… | wǒ... diǎn... | I ... at ... o'clock | productive |
| 先…… | xiān... | first... | productive |
| 然后…… | ránhòu... | then... | productive |
| 你几点……？ | nǐ jǐ diǎn...? | what time do you...? | productive |
| 起床 | qǐchuáng | get up | exposure |
| 吃早饭 | chī zǎofàn | eat breakfast | exposure |
| 上班 | shàngbān | go to work | exposure |
| 学习 | xuéxí | study | exposure |
| 回家 | huí jiā | go home | exposure |
| 睡觉 | shuìjiào | sleep; go to bed | exposure |

## Main task

Describe a morning and evening using three activities, times, and a sequence.

## Changed retry

Describe a weekend morning with one different time and activity.

## Correction priorities

- Put the time before the activity in a complete sentence.
- Keep `然后 ránhòu` as the link between events rather than a separate translation exercise.

## Carry forward

Ask and return one question with `你呢？`.
